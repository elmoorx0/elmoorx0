# @elmoorx/ai-chat

> AI chat: streaming UI, message history, citations, multimodal

Part of the [Elmoorx Framework](https://github.com/elmoorx0/elmoorx0) — Build fast. Run anywhere. Stay secure.

```bash
npm install @elmoorx/ai-chat
```
