# @elmoorx/auth

> Authentication: sessions, JWT, OAuth (Google, GitHub), 2FA, RBAC

Part of the [Elmoorx Framework](https://github.com/elmoorx0/elmoorx0) — Build fast. Run anywhere. Stay secure.

## Installation

```bash
npm install @elmoorx/auth
```

## Quick Start

```typescript
import { /* exports */ } from '@elmoorx/auth';
```

- Zero external dependencies
- Full TypeScript support
- Tree-shakeable
- Edge-runtime compatible
- Arabic/RTL friendly
