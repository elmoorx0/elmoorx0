# @elmoorx/compiler

> JSX-to-Elmoorx compiler with esbuild integration, SSR, HMR

Part of the [Elmoorx Framework](https://github.com/elmoorx0/elmoorx0) — Build fast. Run anywhere. Stay secure.

## Installation

```bash
npm install @elmoorx/compiler
```

## Quick Start

```typescript
import { /* exports */ } from '@elmoorx/compiler';
```

## Features

- Zero external dependencies
- Full TypeScript support
- Tree-shakeable
- Edge-runtime compatible
- Arabic/RTL friendly
