# @elmoorx/css

> Zero-runtime CSS system with atomic classes, theming, RTL support

Part of the [Elmoorx Framework](https://github.com/elmoorx0/elmoorx0) — Build fast. Run anywhere. Stay secure.

## Installation

```bash
npm install @elmoorx/css
```

## Quick Start

```typescript
import { /* exports */ } from '@elmoorx/css';
```

## Features

- Zero external dependencies
- Full TypeScript support
- Tree-shakeable
- Edge-runtime compatible
- Arabic/RTL friendly
