# @elmoorx/devtools

> Browser DevTools extension: signal inspector, time-travel, profiler

Part of the [Elmoorx Framework](https://github.com/elmoorx0/elmoorx0) — Build fast. Run anywhere. Stay secure.

## Installation

```bash
npm install @elmoorx/devtools
```

## Quick Start

```typescript
import { /* exports */ } from '@elmoorx/devtools';
```

## Features

- Zero external dependencies
- Full TypeScript support
- Tree-shakeable
- Edge-runtime compatible
- Arabic/RTL friendly
