# @elmoorx/edge-db

> Edge database: global replication, eventual consistency, low latency reads

Part of the [Elmoorx Framework](https://github.com/elmoorx0/elmoorx0) — Build fast. Run anywhere. Stay secure.

## Installation

```bash
npm install @elmoorx/edge-db
```

## Quick Start

```typescript
import { /* exports */ } from '@elmoorx/edge-db';
```

## Features

- Zero external dependencies
- Full TypeScript support
- Tree-shakeable
- Edge-runtime compatible
- Arabic/RTL friendly
