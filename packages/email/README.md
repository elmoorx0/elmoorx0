# @elmoorx/email

> Email service: SMTP, templates, queue, attachments, tracking

Part of the [Elmoorx Framework](https://github.com/elmoorx0/elmoorx0) — Build fast. Run anywhere. Stay secure.

## Installation

```bash
npm install @elmoorx/email
```

## Quick Start

```typescript
import { /* exports */ } from '@elmoorx/email';
```

- Zero external dependencies
- Full TypeScript support
- Tree-shakeable
- Edge-runtime compatible
- Arabic/RTL friendly
