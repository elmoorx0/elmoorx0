# @elmoorx/forms

> Form library with validation, schemas, async validation, field arrays

Part of the [Elmoorx Framework](https://github.com/elmoorx0/elmoorx0) — Build fast. Run anywhere. Stay secure.

## Installation

```bash
npm install @elmoorx/forms
```

## Quick Start

```typescript
import { /* exports */ } from '@elmoorx/forms';
```

## Features

- Zero external dependencies
- Full TypeScript support
- Tree-shakeable
- Edge-runtime compatible
- Arabic/RTL friendly
