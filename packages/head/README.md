# @elmoorx/head

> SEO head manager: meta tags, Open Graph, Twitter Cards, JSON-LD, favicons

Part of the [Elmoorx Framework](https://github.com/elmoorx0/elmoorx0) — Build fast. Run anywhere. Stay secure.

## Installation

```bash
npm install @elmoorx/head
```

## Quick Start

```typescript
import { /* exports */ } from '@elmoorx/head';
```

## Features

- Zero external dependencies
- Full TypeScript support
- Tree-shakeable
- Edge-runtime compatible
- Arabic/RTL friendly
