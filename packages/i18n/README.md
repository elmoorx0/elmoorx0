# @elmoorx/i18n

> Internationalization: locale switching, RTL support, pluralization, calendars

Part of the [Elmoorx Framework](https://github.com/elmoorx0/elmoorx0) — Build fast. Run anywhere. Stay secure.

## Installation

```bash
npm install @elmoorx/i18n
```

## Quick Start

```typescript
import { /* exports */ } from '@elmoorx/i18n';
```

## Features

- Zero external dependencies
- Full TypeScript support
- Tree-shakeable
- Edge-runtime compatible
- Arabic/RTL friendly
