# @elmoorx/image

> Image optimizer: AVIF/WebP, blur placeholder, responsive srcset

Part of the [Elmoorx Framework](https://github.com/elmoorx0/elmoorx0) — Build fast. Run anywhere. Stay secure.

```bash
npm install @elmoorx/image
```
