# @elmoorx/postgres

> PostgreSQL adapter: connection pooling, transactions, query builder, migrations

Part of the [Elmoorx Framework](https://github.com/elmoorx0/elmoorx0) — Build fast. Run anywhere. Stay secure.

## Installation

```bash
npm install @elmoorx/postgres
```

import { /* exports */ } from '@elmoorx/postgres';
