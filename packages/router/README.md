# @elmoorx/router

> File-system router with dynamic routes, layouts, loaders, code-splitting

Part of the [Elmoorx Framework](https://github.com/elmoorx0/elmoorx0) — Build fast. Run anywhere. Stay secure.

## Installation

```bash
npm install @elmoorx/router
```

## Quick Start

```typescript
import { /* exports */ } from '@elmoorx/router';
```

## Features

- Zero external dependencies
- Full TypeScript support
- Tree-shakeable
- Edge-runtime compatible
- Arabic/RTL friendly
