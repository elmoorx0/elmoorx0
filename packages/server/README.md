# @elmoorx/server

> Full-stack HTTP server with middleware, static files, API routes, WebSocket

Part of the [Elmoorx Framework](https://github.com/elmoorx0/elmoorx0) — Build fast. Run anywhere. Stay secure.

## Installation

```bash
npm install @elmoorx/server
```

## Quick Start

```typescript
import { /* exports */ } from '@elmoorx/server';
```

## Features

- Zero external dependencies
- Full TypeScript support
- Tree-shakeable
- Edge-runtime compatible
- Arabic/RTL friendly
