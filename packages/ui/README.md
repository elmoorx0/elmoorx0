# @elmoorx/ui

> 648-component UI library (forms, layout, nav, data, feedback, media, pro)

Part of the [Elmoorx Framework](https://github.com/elmoorx0/elmoorx0) — Build fast. Run anywhere. Stay secure.

## Installation

```bash
npm install @elmoorx/ui
```

## Quick Start

```typescript
import { /* exports */ } from '@elmoorx/ui';
```

## Features

- Zero external dependencies
- Full TypeScript support
- Tree-shakeable
- Edge-runtime compatible
- Arabic/RTL friendly
