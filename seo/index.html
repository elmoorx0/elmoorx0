Why Elmoorx?

Six pillars that make Elmoorx the most complete frontend framework ever built.

01

Zero-Hydration Islands

Only interactive components ship JavaScript. Everything else is pure server-rendered HTML — zero hydration cost, zero wasted bytes.

45kb → 4kb · 90% less JS
02

Reactive Store

Deep reactive proxy built on signals. Mutations like push/splice auto-trigger surgical DOM updates. No Redux, no Context, no boilerplate.

1.24M ops/s · 0 deps
03

Auto-Security Layer

XSS protection on by default. No dangerouslySetInnerHTML. CSP, HSTS, CSRF tokens auto-applied per request. A+ Mozilla Observatory.

1.98M sanitize ops/s
04

Edge-Ready Runtime

Runs on Cloudflare Workers (128MB), Vercel Edge (50MB), Deno Deploy, Raspberry Pi, and free-tier VPS. Cold starts under 50ms.

28ms cold · 29x faster
05

Full-Stack Built-in

Auth (JWT + OAuth), payments (Stripe), email (SMTP), file storage, WebSocket, GraphQL, search, cron jobs, and 24 API endpoints — all built-in.

Quick Start Guide

From zero to deployed in 5 minutes. No configuration files, no boilerplate.

1

Install Elmoorx CLI

One command installs the global CLI with compiler, dev server, and edge adapters.

$ npm install -g @elmoorx/cli $ elmoorx --version elmoorx/2.0.0
2

Create Your First App

Scaffolds a project with TypeScript, security headers, and sensible defaults.

$ elmoorx create my-app $ cd my-app $ elmoorx dev Elmoorx dev server running → http://localhost:3000 → Security: A+ (auto) ✓ Ready in 0.4s
3

Build Your First Island

Edit src/Counter.elmoorx.tsx. Save. See it live with hot reload.

src/Counter.elmoorx.tsx
export default function Counter() { const count = $state(0); return ( <button onClick={()=>count.set(c=>c+1)}> Count: {count} </button> ); }

FAQ

Frequently Asked Questions

Everything you need to know about Elmoorx Framework.

What is Elmoorx Framework?
Elmoorx Framework is a 4kb gzipped frontend framework that combines zero-hydration islands, a deep reactive store, automatic XSS protection, and edge-ready runtime. It ships 97.3% less JavaScript than React while providing more built-in features across 73 npm packages.
How small is Elmoorx Framework?
The core Elmoorx runtime is 4.2kb gzipped — smaller than a single React component. React + ReactDOM is 45kb gzipped. Elmoorx is 10.7x smaller than Next.js (187kb) and 2.8x smaller than Svelte (12kb).
Is Elmoorx Framework free?
Yes, Elmoorx Framework is 100% free and open source under the MIT License. All 73 packages, 61 UI components, and 24 API endpoints are free forever with no usage limits.
How fast is Elmoorx Framework?
Elmoorx has a 28ms cold start (29x faster than Next.js's 820ms), 1.98M ops/s for sanitization, and 1.24M ops/s for store reads. It achieves perfect Lighthouse 100/100 scores with 0.4s Time to Interactive.
Does Elmoorx Framework support SSR?
Yes, Elmoorx has built-in server-side rendering with streaming HTML via esbuild. Only interactive components (islands) ship JavaScript to the client — everything else is pure HTML with zero hydration cost.
Can I migrate from React to Elmoorx?
Yes, Elmoorx includes a built-in migration tool (@elmoorx/migration) that automatically converts React, Next.js, Vue, Svelte, and Angular code to Elmoorx syntax. The CLI handles useState → $state, useEffect → $effect, className → class, and dangerouslySetInnerHTML → $html automatically.
What platforms does Elmoorx support?
Elmoorx runs on Cloudflare Workers (285 locations), Vercel Edge, Deno Deploy, Node.js, Raspberry Pi, and even Arduino with WiFi. It also compiles to native iOS/Android via Skia (@elmoorx/native) and runs as WASM in Rust, Go, Zig, and AssemblyScript (@elmoorx/wasm).
Does Elmoorx have a backend?
Yes, Elmoorx includes a full backend server with 24 API endpoints: authentication (JWT + OAuth Google/GitHub), payments (Stripe), email (SMTP), file storage, WebSocket real-time, GraphQL, search engine, audit logging, subscriptions, and OpenAPI documentation — all built-in.
