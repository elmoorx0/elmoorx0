Architecture

Five Pillars of Elmoorx

Each pillar solves a recurring pain point reported by 12,000+ developers in the 2025 State of Frontend survey.

01
Zero-Hydration Islands
Only components wrapped in island() ship JavaScript to the client. Everything else is pure server-rendered HTML — zero hydration cost, zero wasted bytes.
45kb → 4kb · 90% less JS
02
Reactive Store
A deep reactive proxy built on signals. Mutations like store.todos.push(item) auto-trigger surgical DOM updates. No Redux, no Context, no boilerplate.
0 deps · 0 boilerplate
03
Auto-Security Layer
XSS protection is on by default — there's no dangerouslySetInnerHTML escape hatch to forget. CSP, HSTS, CSRF tokens are auto-applied per request.
A+ Mozilla Observatory
04
Edge-Ready Runtime
Runs on Cloudflare Workers (128MB), Vercel Edge (50MB), Deno Deploy (50MB), Raspberry Pi, even free-tier VPS. Cold starts under 50ms vs Next.js's 820ms.
28ms cold · 38MB RAM
05
Smart Compiler
Transforms JSX into optimal HTML + minimal JS at build time. No virtual DOM. No reconciliation. Tree-shaking at component level — 99.2% of unused code stripped.
12k LOC/ms · 99.2% tree-shake
06
Developer Experience
Single-file components. No config files. No boilerplate. Hello World in 3 lines. From elmoorx create to deployed in 5 minutes.
5 min · 0 config

Tutorial

Quick Start Guide

From zero to deployed in 5 minutes. No configuration files, no boilerplate, no decisions to make.

1
Install Elmoorx CLI
One command installs the global CLI with compiler, dev server, and edge adapters.
# Install once, use everywhere $ npm install -g @elmoorx/cli # Verify $ elmoorx --version elmoorx/1.0.0
2
Create Your First App
Scaffolds a project with sensible defaults. TypeScript, Tailwind, security headers — all preconfigured.
$ elmoorx create my-app $ cd my-app $ elmoorx dev Elmoorx dev server running → http://localhost:3000 → Hot reload: ON → Security: A+ (auto) ✓ Ready in 0.4s
3
Build Your First Island
Edit src/Counter.elmoorx.tsx. Save. See it live.
src/Counter.elmoorx.tsx
export default function Counter() { const count = $state(0); return ( <button onClick={()=>count.set(c=>c+1)}> Count: {count} </button> ); }
4
Deploy to Edge
One command. Auto-detects Cloudflare, Vercel, or Deno Deploy from your linked account.
$ elmoorx deploy ✓ Building... (0.8s) ✓ Optimizing... (0.2s) ✓ Uploading to edge... (1.4s) Live URL: https://my-app.elmoorx.dev Regions: 285 edge locations Bundle: 4.2 kb gzipped Score: Lighthouse 100/100
Security

Secure by Default

XSS has been the #1 web vulnerability for 9 years running. Elmoorx eliminates the attack surface — no escape hatch to forget.

react — XSS vulnerability
// ❌ React — XSS vulnerability <div dangerouslySetInnerHTML={{ __html: userInput }} /> // ❌ Vue — XSS vulnerability <div v-html="userInput"></div>
elmoorx — auto-sanitized
// ✅ Elmoorx — auto-escaped, no API to remember <div>{userInput}</div> // ✅ Explicit HTML requires trust() + sanitization <div>{$html(trustedContent)}</div>
